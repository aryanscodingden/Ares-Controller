## Ares - A game controller
A custom game controller built using a ESP32. 

<img width="1920" height="654" alt="Game Controller Render 4" src="https://github.com/user-attachments/assets/482959ae-7967-426e-b5f5-089861020504" />

This project was built because I always wanted to own a controller & decided to build one instead. 
It was indeed a very intersting journey to build one of my own, started to appricate the small QOL of other products.
<br>
(P.S. This is my first ever hardware project)

# Schematic
<img width="3507" height="2480" alt="image" src="https://github.com/user-attachments/assets/6a8cea2d-1081-42cb-a628-21c7aaadbdee" />
This schematic was made in KiCAD. It's a very simple controller.
<br>
It features a ESP32-NodeMCU for the main controller board & a TP4056 Module to charge the entire controller.
